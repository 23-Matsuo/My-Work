--起動方法----------------------


「実行可能ファイル」内のJELLY PARTY.exeをダブルクリックでゲーム起動


--遊び方------------------------


お皿の上から落とされないよう回転するバーをジャンプでよけて、
上から降ってくるゼリーを集めてハイスコアを目指せ！


--操作方法----------------------


メニュー操作：

　「W, Sキー」または「矢印キー」	：選択
　「スペースキー」			：決定


ゲーム中の操作：

　「WASDキー」または「矢印キー」	：移動
　「スペースキー」			：ジャンプ
